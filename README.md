# _nix-aura
A porting project of the aura-2 project into *nix-like OSs using D and C/C++ languages.
- Original source: https://github.com/elfinlazz/aura-2.git

## Requirements
- cmake 3.10 or higher
- C/C++ compiler (C99/C++17)
- D v2
- pthreads
- gcrypt

## Installation
```
mkdir build;
cd build;
cmake ../ ;
make
```
